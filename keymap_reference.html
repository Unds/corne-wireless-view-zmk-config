Combos
Active on Layer 0

Press both keys simultaneously to trigger the combo output. All combos are defined on the base layer and use urob's ZMK_COMBO helper.

Combo Definitions
Keys (press together) Output Position on board
Q + W ESC Top row, left — positions LT1 + LT2
W + F TAB Top row, left — positions LT2 + LT3
T + N Caps Word Home row, both hands — positions LM3 + RM3
U + Y DEL Top row, right — positions RT3 + RT4
D + V Win+Shift+S Screenshot Bottom row, left — positions LB3 + LB4
Left-middle thumb + Right-middle thumb Toggle MapleStory (Layer 7) Thumbs — positions LH1 + RH1
Caps Word

Caps Word capitalises all letters until a word-breaking key is pressed (space, enter, punctuation other than - and _). Ideal for typing CONSTANTS_LIKE_THIS without holding Shift. Triggered by the T + N combo.

Sticky Shift

The left Shift key (skq LSHFT) is configured as a sticky key with a 1000 ms release window and quick-release. Tap once → shift applies to the very next key only. No need to hold. Ideal for capitalising single characters.
